### Guardian Angel
Guardian angel fetches its user’s health data, and determines their physical health using the
data given.

### Objectives
- Help the person maintain a healthy diet
- Help the person adjust their lifestyle to be healthy
- Identify whether or not they have a disease, and detect what it is

### Features
- Health Diet Tracker
- Disease Determiner
- Reliability Verifier (Addition to the Disease Determiner)
- Health Monitor

### Prerequisites
- IDLE 3.13.3 or Standard Python

### Installation and Running of Program
1. Install Python
2. Download Program/Code from Github
